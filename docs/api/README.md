# API Documentation Overview

This folder contains the consolidated API reference for the **get‑stockdata** service. The documentation is organized by EPIC, with each EPIC having its own sub‑folder:

- `epic005/` – Multi‑Level Stock Pool Data APIs (real‑time quotes, liquidity, status, fundamentals, enhanced stock list).
- `epic002/` – Fundamental & Valuation Data APIs (financial indicators, valuation, industry statistics, enhanced stock info).

The master index (`index.md`) provides quick navigation to each EPIC's API reference. For error handling details, see the **Error Codes** reference.
