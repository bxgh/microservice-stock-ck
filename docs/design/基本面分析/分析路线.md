公司产业定位与行业地位分析模板
| 模块 | 关键指标 | 数据来源/说明 | 分析要点 |
|------|----------|----------------|----------|
| 1. 公司基本信息 | 公司名称、主营业务、注册地、上市板块| 年报、官网、证监会公告 | 明确公司所处行业和核心产品线|
| 2. 行业与产业链位置 | 行业分类（一级/二级）、产业链环节（上游/中游/下游/服务） | 年报主营业务、招股书、行业协会| 判断公司在产业链中的位置与作用 |
| 3. 市场份额与规模| 营收总额、产量/销量、市场占比、客户集中度 | 行业统计、协会数据、年报前五大客户 | 高份额 → 龙头；低份额 → 小玩家；客户集中度判断依赖性 |
| 4. 技术与产品竞争力 | 研发投入占比、专利数量与类型、核心技术/认证 | 年报研发报告、专利数据库、产品认证 | 高研发 + 专利多 → 技术壁垒高；认证/奖项 → 产品竞争力 |
| 5. 财务表现 | 毛利率、净利率、ROE、ROA、现金流 | 年报、财报摘要 | 高毛利率/ROE → 资本效率高；稳定现金流 → 行业地位稳固 |
| 6.竞争格局 | 行业集中度CR3/CR5、主要竞争对手对比| 行业协会、Wind、东方财富 | 低集中度 →竞争激烈；高集中度 → 龙头优势明显 |
| 7. 政策与资源依赖 | 政策扶持、牌照资质、合作伙伴/客户结构 | 政府公告、年报、招股书 | 高政策依赖 → 行业地位受政策影响；核心客户 → 行业话语权 |
| 8. 综合评价 | 行业地位总结 | 前述数据分析 | 定性+定量结合：龙头/核心玩家/成长型/跟随者 |---
使用方法

1.收集数据：从年报、招股书、行业协会数据、市场研究报告收集上述各项指标。
2.填入表格：根据数据填写每一列内容。
3.定量评估：通过营收占比、市场份额、毛利率等指标初步判断地位。
4.定性分析：结合技术、政策、客户资源等因素进行综合判断。
5.输出结论：最后形成一段文字总结公司在产业链中的核心地位及竞争力。
