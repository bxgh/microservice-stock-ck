# 资金获取方案: 量化基金

## 底层数据
量化基金数据需通过“衍生指标”反推。

## 计算公式 (四因子合成)
```
量化活跃度 = 0.3 × 指增超额Score + 0.25 × 小微盘强度Score + 0.25 × 期货贴水Score + 0.2 × 成交集中度Score
```

### 1. 指增超额 Score
- **逻辑**: 中证500/1000指增滚动的20日超额收益。
- **数据源**: 好买基金网、私募排排网; 或代表性产品 (如幻方、明汯) 的净值。

### 2. 小微盘强度 Score
- **逻辑**: 微盘股指数 (8841431) / 中证1000指数 (000852) 的相对强度。
- **计算**: `Ratio = 微盘股 / 中证1000`, `Score = (今日Ratio - 60日均值) / 60日标准差`。

### 3. 期货贴水 Score
- **逻辑**: IC (中证500期货) 和 IM (中证1000期货) 主力合约的年化贴水率。
- **计算**: `贴水率 = (现货 - 期货) / 期货 × (252/剩余天数)`。
- **数据源**: 中金所、Wind、Choice。

### 4. 成交集中度 Score
- **逻辑**: 中证500和中证1000成分股占全A成交额的比例。
- **指示意义**: 比例上升代表量化交易活跃。

# 1. 活跃度识别分 (用于主导资金识别, 20日窗口)
ActivityScore = (今日量化活跃度 - 20日均值) / 20日标准差 × Continuity(5d)

# 2. 状态屏温度计得分 (用于视觉显示, 60日窗口)
Quant_Score = Z(量化活跃度, 60d)

## 量化拥挤度警戒灯 (状态屏"量化拥挤度"指标)
- **因子合成**: 0.4 × 指增超额z + 0.3 × 小微盘强度z + 0.3 × 期货贴水z
- **判定规则**:
    - **绿灯**: 综合分位 < 60% (安全, 量化策略尚有容量)
    - **黄灯**: 60% ≤ 综合分位 ≤ 85% (拥挤, 策略波动可能加大)
    - **红灯**: 综合分位 > 85% (极度拥挤, 存在流动性负反馈风险)
