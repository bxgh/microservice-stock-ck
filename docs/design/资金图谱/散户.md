# 资金获取方案: 散户

## 底层数据
多源合成, 涵盖杠杆、成交明细、开户及关注度。

## 计算公式 (四因子合成)
```
散户活跃度 = 0.3 × 两融Score + 0.25 × 小单占比Score + 0.25 × 新增开户Score + 0.2 × 搜索指数Score
```

### 1. 两融 Score
- **数据**: 两融余额日变化 / 前一日余额。
- **计算**: `Score = (今日变化率 - 30日均值) / 30日标准差`。

### 2. 小单占比 Score
- **数据**: 全 A 股票小单 (<5 万元) 成交额占比。
- **来源**: 交易所 L2 数据, 或同花顺/东财。

### 3. 新增开户 Score
- **数据**: 中证登周度开户数据, 日度线性插值。
- **计算**: `Score = (本周新增 - 12周均值) / 12周标准差`。

### 4. 搜索指数 Score
- **数据**: 百度指数中“股票”、“开户”、“炒股”的搜索量。
- **计算**: `Score = (今日指数 - 30日均值) / 30日标准差`。

# 1. 活跃度识别分 (用于主导资金识别, 20日窗口)
ActivityScore = (今日散户活跃度 - 20日均值) / 20日标准差 × Continuity(5d)

# 2. 状态屏温度计得分 (用于视觉显示, 30日/60日窗口)
Retail_Score = Z(散户活跃度, 30d)

## 散户情绪警戒灯 (状态屏"散户情绪"指标)
- **因子合成**: 0.4 × 两融z + 0.3 × 新增开户z + 0.3 × 搜索指数z (剔除小单占比, 追求宏观情绪)
- **判定规则**:
    - **绿灯 (低频)**: 综合 Z 在 [-1, +1]
    - **黄灯 (过热/过冷)**: 综合 Z 在 [-2, -1] 或 [+1, +2]
    - **红灯 (极致情绪)**: 综合 Z > +2 (极端贪婪) 或 < -2 (极端恐惧)

## 重要洞察 (反向指标)
散户活跃度往往是 **反向指标**。系统应在红灯极致值时自动弹出“反向警示”。
