# 资金获取方案: 险资

## 底层数据
险资高频数据极难获取, 主要依赖月度披露和事件驱动。

## 计算逻辑
```
日常温度计 = 月度险资增减仓 / 22交易日 (月度数据线性平滑)
事件驱动 = 险资举牌公告、大宗增减持公告 (即时更新)

最终得分 = 日常温度计得分 + 事件驱动加权得分
```

## 数据源
- 国家金融监管总局月度数据 (月初发布上月)。
- 上市公司险资举牌公告 (即时)。
- 中保协人寿险公司投资数据。

## 特殊提示
险资温度计在大部分时间会显示为“近中性”, 因为月度数据高度平滑。但在出现险资举牌时, 温度计应立即跳到强正值并附弹出说明。
