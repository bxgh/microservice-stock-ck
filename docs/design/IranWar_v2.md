# 地缘冲突防御型选股策略 v2.0

> 基于原 `IranWar.md` 需求重新设计，适配现有微服务数据能力。

---

## 一、原始策略问题诊断

| 问题 | 影响 |
|---|---|
| 参数"N个交易日"未定义 | 无法确定暴跌判定窗口 |
| 毛利率 >30% / ROE >15% 跨行业一刀切 | 会误杀银行（毛利概念不适用）、误选周期股（ROE波动大） |
| "利润加速"依赖季报（滞后1-4个月） | 实时性差，在地缘事件驱动场景中几乎失效 |
| "全球定价权"映射为毛利率和ROE | 衡量的是盈利能力而非定价权 |
| 缺少入场/退出/仓位规则 | 只有选股没有交易，无法闭环 |
| 要求用 Tushare/AkShare | 与现有微服务架构脱节 |

---

## 二、重新定义策略目标

**核心假设**：地缘冲突（如伊朗战争）导致全球风险资产抛售 → A股大盘短期暴跌 → 但具备"国家安全"属性、基本面扎实、资金结构稳固的个股会率先企稳并在恐慌平息后领涨。

**策略定位**：**事件驱动型防御策略**——在系统性风险事件发生后，根据冲突的严重程度和持续时间，切换不同的选股逻辑和持仓方案。

**策略不是**：
- 不是预测地缘事件的工具
- 不是自动交易策略（产出候选池，人工决策入场）

### 2.1 本次战争实例参数

| 参数 | 值 | 说明 |
|---|---|---|
| **冲突开始日** | **2026-03-01** | 以此为 T=0 计算冲突持续天数 |
| **当前状态** | 情景 A（闪击战） | 截至 2026-03-05，冲突持续 3 个交易日，未触发升级条件 |
| **升级监控** | 冲突 > 10 交易日 且 WTI 涨幅 > 20% → 升级到情景 B | 预计最早判定日：2026-03-17 左右 |
| **油价基准** | 2026-02-28 收盘价作为基准 | 用于计算油价涨幅百分比 |

## 三、战争持续时间情景分析

### 3.1 历史参考

历次中东战争对市场的影响存在明确的"持续时间-冲击烈度"规律：

| 历史事件 | 持续时间 | 油价影响 | A股/全球股市 | 黄金 |
|---|---|---|---|---|
| 海湾战争 (1991) | 地面战仅100小时 | 3个月内从$17→$41，开战后迅速回落至$20 | 美股熊市仅11周，随后一年涨25% | "买预期卖事实"，开战当日暴跌5.3% |
| 伊拉克战争 (2003) | 主要战斗3周 | 战前冲高，开战后回落 | 战前下跌，开战后反弹 | 战前上涨，开战后回调 |
| 也门危机 (2015-至今) | 持续数年 | 对油价影响有限（未封锁航道） | A股几乎无反应 | 无显著独立影响 |

**核心规律**：
1. A股对中东冲突的敏感度远低于欧美市场，但**若冲突波及霍尔木兹海峡（全球1/5石油运输通道），影响将非线性放大**
2. 冲突越短，市场越倾向"V型反转"；冲突越长，越会从情绪冲击转向基本面重估
3. 黄金在开战前上涨、开战后回落是高概率事件

### 3.2 三种情景定义

```
                    伊朗战争持续时间情景图

    情景A: 闪击战          情景B: 中等冲突         情景C: 持久战
    ─────────           ──────────────         ──────────
    1-2 周               1-3 个月                3个月以上

    定点打击              持续空袭               地面入侵
    未封锁航道            航道受扰               航道封锁
    油价脉冲<20%          油价上涨20-50%          油价翻倍+

    A股: 1-3日恐慌        A股: 持续承压2-4周      A股: 系统性重估
    → V型反转             → U型修复              → L型底部
```

### 3.3 情景 A：闪击战（1-2 周）

**假设前提**：外科手术式打击伊朗核设施/军事目标，未引发全面战争，霍尔木兹海峡未被封锁。

**市场预期**：A股 1-3 个交易日恐慌下跌，随后快速收复。油价脉冲上涨 10-20% 后回落。

| 参数 | 设定值 | 说明 |
|---|---|---|
| **大盘触发条件** | 3日沪深300跌幅 ≥ 3% | 短窗口、低阈值，快速捕捉脉冲 |
| **选股观察窗口** | 3 个交易日 | 与触发窗口一致 |
| **持仓周期** | 5-10 个交易日 | V型反弹的时间窗口 |
| **止盈目标** | 超额收益 8% 或绝对收益 5% | 短线快进快出 |
| **止损线** | 超额收益 -5% | 情景判断错误时及时退出 |
| **市值门槛** | > 100 亿 | 短线流动性要求更高 |
| **池子大小** | Top 10 | 集中持仓，捕捉反弹弹性 |

**行业侧重**：

| 优先级 | 行业 | 逻辑 |
|---|---|---|
| **核心** | 军工（航空航天/地面兵装） | 地缘事件直接受益，脉冲弹性最大 |
| **核心** | 贵金属（黄金） | 避险情绪短期驱动 |
| **辅助** | 油气开采/油服 | 油价脉冲的直接映射 |

**评分权重调整**（相对 v2.0 基准）：

| 维度 | 基准权重 | 闪击战权重 | 理由 |
|---|---|---|---|
| 抗跌能力（超额收益） | 30% | **40%** | 短线反弹中，前期抗跌股弹性最大 |
| 缩量抗跌 | 15% | 10% | 窗口太短，换手率数据参考价值下降 |
| 概念板块 | 10% | **15%** | 军工/黄金概念短期情绪加成更强 |
| 国家队持仓 | 15% | 10% | 短线博弈中国家队影响权重降低 |
| 北向资金 | 10% | 10% | 保持 |
| 基本面前瞻 | 10% | 5% | 短线交易中基本面权重降低 |
| 估值安全边际 | 10% | 10% | 保持 |

### 3.4 情景 B：中等冲突（1-3 个月）

**假设前提**：持续空袭 + 有限升级，霍尔木兹海峡通行受扰但未完全封锁，伊朗可能进行有限报复（袭击沙特油田/以色列）。

**市场预期**：A股 2-4 周持续承压，行业分化剧烈（能源/军工上涨，消费/科技承压）。油价上涨 20-50%。

| 参数 | 设定值 | 说明 |
|---|---|---|
| **大盘触发条件** | 5日跌幅 ≥ 5% 或 20日跌幅 ≥ 8% | 标准条件（v2.0 原始设定） |
| **选股观察窗口** | 10-20 个交易日 | 需要足够长的窗口验证抗跌持续性 |
| **持仓周期** | 15-40 个交易日 | U型修复的时间窗口 |
| **止盈目标** | 超额收益 15% 或绝对收益 10% | 中线波段 |
| **止损线** | 超额收益 -8% 或绝对收益 -10% | 容错空间更大 |
| **市值门槛** | > 50 亿 | 标准设定 |
| **池子大小** | Top 20 | 适度分散 |

**行业侧重**：

| 优先级 | 行业 | 逻辑 |
|---|---|---|
| **核心** | 油气开采、煤炭开采 | 能源价格持续走高，业绩直接受益 |
| **核心** | 军工全产业链 | 持续冲突 → 军费预期上行 |
| **核心** | 贵金属、小金属、工业金属 | 避险+供应链扰动双重驱动 |
| **辅助** | 农药/种子 | 粮食安全预期（中东局势影响全球粮价） |
| **辅助** | 半导体/通信设备 | 自主可控预期升温 |

**评分权重调整**：

| 维度 | 基准权重 | 中等冲突权重 | 理由 |
|---|---|---|---|
| 抗跌能力（超额收益） | 30% | 30% | 保持核心地位 |
| 缩量抗跌 | 15% | 15% | 窗口够长，筹码结构有意义 |
| 概念板块 | 10% | 10% | 保持 |
| 国家队持仓 | 15% | 15% | 保持 |
| 北向资金 | 10% | **5%** | 中等冲突中北向可能持续流出，参考价值降低 |
| 基本面前瞻 | 10% | **15%** | 冲突持续时基本面变得重要（能源股业绩兑现） |
| 估值安全边际 | 10% | 10% | 保持 |

### 3.5 情景 C：持久战（3 个月以上）

**假设前提**：全面战争，地面入侵，霍尔木兹海峡被封锁或高度危险，伊朗动用导弹攻击中东油田，美国全面介入。全球能源格局重塑。

**市场预期**：A股系统性重估，持续下行后进入 L 型底部。油价翻倍以上。全球滞涨风险升级。

| 参数 | 设定值 | 说明 |
|---|---|---|
| **大盘触发条件** | 20日跌幅 ≥ 10% 且 60日跌幅 ≥ 15% | 高阈值确认系统性下行 |
| **选股观察窗口** | 30-60 个交易日 | 需要长窗口验证"真抗跌" |
| **持仓周期** | 60+ 个交易日（至冲突降级信号出现） | 长期防御持仓 |
| **止盈目标** | 不设固定止盈，跟踪止盈（从最高点回撤 10% 平仓） | 趋势跟踪而非固定目标 |
| **止损线** | 绝对收益 -15% | 系统性风险下容忍更大回撤 |
| **市值门槛** | > 200 亿 | 极端环境下只选大票 |
| **池子大小** | Top 15 | 精选+分散 |

**行业侧重（重大变化）**：

| 优先级 | 行业 | 逻辑 |
|---|---|---|
| **核心** | 煤炭开采 | 能源替代：油价暴涨 → 煤化工/煤制油经济性飙升 |
| **核心** | 电池/光伏设备 | 能源转型加速：高油价倒逼新能源建设提速 |
| **核心** | 贵金属 | 长期避险资产 |
| **核心** | 军工全产业链 | 长期军费上行确定性最高 |
| **辅助** | 特高压/电力 | 能源安全基础设施 |
| **辅助** | 种子/农药 | 粮食安全（全球粮价可能受冲击） |
| **排除** | 消费/白酒/旅游 | 滞涨环境下消费承压 |
| **排除** | 航空/航运 | 油价暴涨直接侵蚀利润 |

**评分权重调整**：

| 维度 | 基准权重 | 持久战权重 | 理由 |
|---|---|---|---|
| 抗跌能力（超额收益） | 30% | **20%** | 长期下行中所有股都会跌，抗跌相对性降低 |
| 缩量抗跌 | 15% | **20%** | 筹码锁定在持久战中更重要（谁在持有？） |
| 概念板块 | 10% | 5% | 长期看概念炒作衰退，回归基本面 |
| 国家队持仓 | 15% | **20%** | 极端环境下国家队托底是核心安全垫 |
| 北向资金 | 10% | 0% | 持久战中北向大概率持续流出，不参考 |
| 基本面前瞻 | 10% | **20%** | 长期持仓必须有基本面支撑 |
| 估值安全边际 | 10% | **15%** | 下跌空间有限的低估值股更抗风险 |

**额外规则**：
- 新增**行业排除名单**（消费/航空/航运），这些行业在持久战中基本面确定性恶化
- 新增**股息率筛选**：要求过去12个月股息率 > 2%（防御属性）
- 新增**大股东增持信号**：近30天产业资本增持 → 强加分

### 3.6 情景切换机制

策略启动后默认进入情景 A（闪击战），根据以下信号自动升级：

```
情景 A（闪击战）
    │
    ├─ 冲突持续 > 10 个交易日 且 油价涨幅 > 20%
    │   → 升级到 情景 B（中等冲突）
    │
    └─ 冲突在 10 个交易日内结束
        → 维持情景 A，到期平仓

情景 B（中等冲突）
    │
    ├─ 冲突持续 > 40 个交易日 且 沪深300累计跌幅 > 15%
    │   → 升级到 情景 C（持久战）
    │
    └─ 冲突降级信号出现（停火谈判/油价回落20%）
        → 降级到 情景 A 的退出逻辑

情景 C（持久战）
    │
    └─ 停火/和平协议 或 油价从峰值回落 30%
        → 启动退出流程（20交易日内逐步减仓）
```

**情景切换时的操作**：
1. 按新情景的行业侧重，调整持仓结构（卖出不在新行业白名单中的标的）
2. 按新情景的评分权重，重新排序并更新 Top N
3. 按新情景的止盈/止损参数，更新风控阈值

### 4.1 原油期货数据采集方案

**数据源**：AkShare `futures_foreign_hist` 接口（数据来源：新浪财经）

| 品种 | AkShare symbol | 用途 |
|---|---|---|
| WTI 原油 | `CL` | 主要参考（全球定价基准） |
| Brent 原油 | `OIL` | 辅助参考 |

**采集方案**：

| 项 | 方案 |
|---|---|
| **采集频率** | 每日 1 次（A股收盘后，北京时间 16:00） |
| **采集字段** | `date`, `open`, `high`, `low`, `close`, `volume` |
| **存储** | ClickHouse 新建 `futures_kline_daily` 表 |
| **实现方式** | 通过 `mootdx-source` 的 AkShare 通道调用，或在 `altdata-source` 中新增采集任务 |
| **历史回填** | 回填最近 1 年数据，用于计算基准价格和波动率 |

**ClickHouse 表设计**：

```sql
CREATE TABLE stock_data.futures_kline_daily_local (
    symbol String,           -- 品种代码（CL/OIL）
    trade_date Date,         -- 交易日期
    open_price Float64,
    high_price Float64,
    low_price Float64,
    close_price Float64,
    volume UInt64,
    update_time DateTime DEFAULT now()
) ENGINE = ReplacingMergeTree(update_time)
PARTITION BY toYYYYMM(trade_date)
ORDER BY (symbol, trade_date);
```

**油价涨幅计算**：

```
油价涨幅(%) = (当日WTI收盘价 - 2026-02-28 WTI收盘价) / 2026-02-28 WTI收盘价 × 100
```

## 五、策略设计（三层漏斗）

### 第一层：市场环境判定 + 抗跌股筛选（详细逻辑见第三章各情景参数）

#### 5.1.1 "大盘暴跌"的量化定义（基准条件，各情景有调整）

使用两个条件联合触发，避免单一参数的任意性：

| 参数 | 定义 | 数据来源 |
|---|---|---|
| 短期急跌 | 过去 **5 个交易日**，沪深300指数累计跌幅 ≥ **5%** | `stock_kline_daily` (code=`000300`) |
| 中期下行确认 | 过去 **20 个交易日**，沪深300累计跌幅 ≥ **8%** | 同上 |

> 两个条件满足其一即触发。5日/5%捕捉急跌事件，20日/8%捕捉持续性调整。

#### 5.1.2 个股抗跌能力量化

在触发"大盘暴跌"的时间窗口内，对全市场筛选：

| 指标 | 计算方式 | 筛选条件 | 数据来源 |
|---|---|---|---|
| **超额收益** | 个股区间收益率 - 沪深300区间收益率 | > **0%** （即跑赢大盘） | `stock_kline_daily` |
| **最大回撤控制** | 个股在窗口内的最大回撤 | < 大盘最大回撤 × **0.7** | `stock_kline_daily` |
| **缩量特征** | 窗口内日均换手率 / 前60日日均换手率 | < **1.2** （未出现恐慌性放量抛售） | `stock_kline_daily` (turnover_rate) |

超额收益是核心指标；最大回撤控制排除"先涨后暴跌"的假抗跌股；缩量特征确认抗跌是因为卖盘枯竭（筹码锁定），而非对倒拉升。

### 第三层：资金结构与基本面验证

#### 5.3.1 资金面验证（使用现有数据）

| 指标 | 计算方式 | 筛选条件 | 数据来源 |
|---|---|---|---|
| **"国家队"持仓** | 前十大股东中是否包含中央汇金、证金公司、社保基金 | 包含 → 加分 | `stock_top_holders` (holder_name 关键词匹配) |
| **北向资金态度** | 暴跌期间北向持仓变化 | 增持 → 加分，大幅减持 → 减分 | `stock_north_funds_daily` |
| **大宗交易信号** | 暴跌期间大宗交易折价率 | 折价 < 5% 或溢价 → 加分（说明有人主动接盘） | `stock_block_trade` |

#### 5.3.2 基本面底线（复用现有能力）

**直接复用** `quant-strategy` 的 `FundamentalFilter`，执行 6 项风控规则：
- ST 风险排除
- 监管黑名单排除
- 商誉占净资产比 < 30%
- 大股东质押率 < 50%
- 收现比 > 0.5
- 财务造假风险排除

**新增估值约束**（使用 `stock_valuation`）：

| 指标 | 条件 | 说明 |
|---|---|---|
| 市值 | > 50 亿 | 排除小盘壳股，确保流动性 |
| PE（市盈率） | > 0 且 < 80 | 排除亏损股和极端高估值 |

#### 5.3.3 替代原文档中的"全球定价权"和"利润加速"

原文档的这两个指标在现有数据下无法准确量化（毛利率不等于定价权、季报滞后严重）。用以下替代方案：

| 原指标 | 替代方案 | 理由 |
|---|---|---|
| 全球定价权（毛利率>30%, ROE>15%） | **业绩预告加分**：最新一期业绩预告类型为"预增"或"略增" | `stock_performance_forecast` 比正式财报时效更高，且是前瞻性信号 |
| 利润加速（季度环比增速） | **分析师评级加分**：最近30天内有分析师给出"买入"或"增持"评级 | `stock_analyst_rank` 数据比季报更及时，反映市场预期 |

## 六、评分与排序（基准权重，各情景有调整见第三章）

通过三层漏斗后的股票，按以下评分体系排序：

| 维度 | 指标 | 权重 | 计分规则 |
|---|---|---|---|
| **抗跌能力** | 超额收益 | 30% | 线性映射到 0-100 |
| **抗跌质量** | 缩量抗跌比 | 15% | 比值越小（越缩量）分越高 |
| **政策加持** | 概念板块命中数 | 10% | 每命中一个加分概念 +20，上限100 |
| **资金认可** | 国家队持仓 | 15% | 有=100, 无=0 |
| **资金态度** | 北向资金变化 | 10% | 增持=100, 持平=50, 减持=0 |
| **基本面前瞻** | 业绩预告 + 分析师评级 | 10% | 有正面信号=100, 无=50 |
| **估值安全边际** | PE 分位数（近3年） | 10% | 越低分越高 |

总分 = 各维度加权和。按总分降序排列，取 Top 20 构成最终防御型股票池。

## 八、与 tiandao.md 的协同关系

`tiandao.md` 定义的是**宏观层面的系统性风险遥测**（美债利差、SOFR异常、金价剪刀差等），其输出是"地缘/宏观风险等级"信号。

本策略可以作为 `tiandao` 的**下游消费者**：
- 当 `tiandao` 输出"高风险"信号 → 自动触发本策略的"大盘暴跌"环境判定
- 当 `tiandao` 未部署时 → 本策略使用自身的沪深300跌幅判定独立运行

两者是**上下游关系**，非并列关系。
