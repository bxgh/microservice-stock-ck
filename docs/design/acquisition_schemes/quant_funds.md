# 资金获取方案: 量化基金

## 底层数据
量化基金数据需通过“衍生指标”反推。

## 计算公式 (四因子合成)
```
量化活跃度 = 0.3 × 指增超额Score + 0.25 × 小微盘强度Score + 0.25 × 期货贴水Score + 0.2 × 成交集中度Score
```

### 1. 指增超额 Score
- **逻辑**: 中证500/1000指增滚动的20日超额收益。
- **数据源**: 好买基金网、私募排排网; 或代表性产品 (如幻方、明汯) 的净值。

### 2. 小微盘强度 Score
- **逻辑**: 微盘股指数 (8841431) / 中证1000指数 (000852) 的相对强度。
- **计算**: `Ratio = 微盘股 / 中证1000`, `Score = (今日Ratio - 60日均值) / 60日标准差`。

### 3. 期货贴水 Score
- **逻辑**: IC (中证500期货) 和 IM (中证1000期货) 主力合约的年化贴水率。
- **计算**: `贴水率 = (现货 - 期货) / 期货 × (252/剩余天数)`。
- **数据源**: 中金所、Wind、Choice。

### 4. 成交集中度 Score
- **逻辑**: 中证500和中证1000成分股占全A成交额的比例。
- **指示意义**: 比例上升代表量化交易活跃。
