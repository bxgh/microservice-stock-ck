# 资金获取方案: 游资

## 底层数据
龙虎榜每日席位数据 + 自维护的游资席位库。

## 计算公式 (三因子合成)
```
游资活跃度 = 0.4 × 上榜席位数Score + 0.3 × 净买入金额Score + 0.3 × 题材集中度Score
```

### 1. 上榜席位数 Score
- **逻辑**: 当日龙虎榜中, 在自维护游资席位库内的席位数量。
- **计算**: `Score = (今日席位数 - 30日均值) / 30日标准差`。

### 2. 净买入金额 Score
- **逻辑**: 所有库内游资席位的净买入合计。
- **计算**: `Score = (今日净买入 - 30日均值) / 30日标准差`。

### 3. 题材集中度 Score
- **逻辑**: 当日上榜个股归属题材的集中程度。题材越少, 反映主线越明确。

## 游资席位库的维护
这是一个核心私有数据资产:
- **初始化**: 从知名网站收集 50-100 个核心席位。
- **持续维护**: 每月 Review 上榜频次, 更新活跃状态。
- **标签系统**: 标注偏好题材、风格、资金规模。

## 数据源
- 上交所龙虎榜: `http://www.sse.com.cn/disclosure/diclosure/public/`
- 深交所龙虎榜: `http://www.szse.cn/disclosure/diclosure/`
- 接口: akshare 的 `stock_lhb_detail_em()`。
