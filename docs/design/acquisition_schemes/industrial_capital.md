# 资金获取方案: 产业资本

## 底层数据
上市公司大股东和董监高的增减持公告。

## 计算公式
```
产业资本活跃度 = 增持金额合计 - 减持金额合计 
净额Score = (今日净额 - 60日均值) / 60日标准差
```

## 数据源
- 上市公司“重大股东持股变动”公告。
- akshare 的 `stock_em_dxsyl()` (大宗交易)、`stock_dzjy_yybph()` (营业部排名)。
- Choice 的“大股东增减持”接口。

## 特殊提示 (领先指标)
产业资本的信号往往是 **领先指标**。
- 当某个行业出现连续多家公司大股东增持时, 通常是该行业基本面信号的拐点。
- **功能建议**: 系统应按行业维度聚合显示, 标注“今日增持最多的 3 个行业”。
