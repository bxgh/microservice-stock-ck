### 【系统开发指令：天道相变遥测矩阵 v1.0】

**一、 系统终极物理目标**
构建一套脱离主观叙事的绝对量化监控管道。系统需实时测度全球宏观系统的“热力学耗散极值”与“物理摩擦力”，在系统状态机发生不可逆相变前，向交易中枢输出机器可读的绝对熔断信号。

**二、 模块拆解与物理传感器部署**

#### 模块 A：外部熵增源监控（美国系统热力学死锁）

* **数据源接口：** FRED (Federal Reserve Economic Data) API, Bloomberg Terminal API。
* **核心传感器与计算算子：**
1. **债务热耗散率：** 提取联邦利息支出（A091RC1Q027SBEA）与联邦税收总额。计算比值 $R_{debt} = \frac{Interest\_Expense}{Tax\_Revenue}$。设定硬性物理阈值报警线（如 $R_{debt} \geq 35\%$）。
2. **重力反转斜率（熊陡测度）：** 提取10年期（T10Y2Y）与2年期美债收益率。当倒挂解除（$Spread > 0$）且呈现二阶加速上行趋势 $\frac{d(Spread)}{dt} > 0$ 时，输出状态相变信号。
3. **微观摩擦力极值：** 高频提取 SOFR (Secured Overnight Financing Rate) 日度数据。计算滚动30日标准差 $\sigma$。当单日读数偏离均值突破 $+3\sigma$ 时，触发“流动性物理休克”最高级警报。



#### 模块 B：跨域阻隔网监控（物理绝缘与能量转移）

* **数据源接口：** 上海黄金交易所 (SGE) API, 央行公开数据接口, COMEX API。
* **核心传感器与计算算子：**
1. **高熵隔离测度（金价剪刀差）：** 同步提取离岸人民币计价黄金（AU9999_CNH）与美元计价黄金（XAU_USD）的波动率。当外部危机爆发，美元金价大幅波动而人民币金价维持低波动，输出“系统物理隔离成功”信号。
2. **底层洋流加速度：** 提取央行“跨境人民币收付金额”月度数据。强制进行多项式拟合，计算其二阶导数 $a_{RMB} = \frac{d^2(Settlement)}{dt^2}$。当 $a_{RMB}$ 突破历史极值，确认全系统“底层物理本位”重置。



#### 模块 C：内部受体池监控（A股状态机强制重置）

* **数据源接口：** Wind API / Tushare Pro / 万得量化接口。
* **核心传感器与计算算子：**
1. **生存配额触发器：** 提取沪深300指数股息率（$Yield_{CSI300}$）与10年期中国国债收益率（$Yield_{CGB10Y}$）。计算差值 $\Delta = Yield_{CSI300} - Yield_{CGB10Y}$。当 $\Delta$ 触及历史 $99\%$ 分位数极值，输出“绝对物理安全垫形成”指令。
2. **地道资产收敛斜率：** 建立“硬核物理底座”自定义股票池（涵盖煤炭、特高压、铁路港口龙头）。计算该组合相对于万得全A指数的相对强度（Relative Strength, RS）。当大盘净流出而该组合 RS 呈现非线性飙升时，确认资金的“强制防守收敛”。
3. **机械踩踏测度：** 监控外资重仓Top 50白马股的每日大宗交易折价率（Discount Rate）与换手率。排除基本面公告日，若出现无差别、高折价的集中抛售，标记为“外部高熵传导导致的机械平仓”，输出左侧承接信号。



**三、 信号降噪与执行输出协议**

1. **底层数据库强制要求：** 全量数据废弃传统关系型数据库，必须写入时序数据库（InfluxDB 或 DolphinDB），以支持毫秒级时序对齐与极端行情下的高并发写入。
