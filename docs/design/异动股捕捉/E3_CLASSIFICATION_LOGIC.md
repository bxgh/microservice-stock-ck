## E3-S1 4 类 (C1-C4) 单标签判定函数

**作为** 17:30 评分流程, **我希望** 给每只异动股打一个机制分类标签, **以便** 评估系统能按机制分组诊断。

#### 任务
- [x] E3-S1-T1 实现分类函数 (Python)
- [x] E3-S1-T2 在 17:30 评分流程中插入分类逻辑
- [x] E3-S1-T3 写入主表 `anomaly_category` 字段

#### 分类优先级 (命中即停)
1. **C4 事件驱动**: 当日有重要事件 (增减持/回购/业绩等)。
2. **C2 中线连板/题材**: `board_height >= 3` 且无事件。
3. **C3 资金驱动**: LHB 上榜 或 北向资金显著异常。
4. **C1 短线封板/普通异动**: 兜底类别。

#### 验收标准 (AC)
- **AC1: 分类覆盖完整**
  - **Then** 100% 的标的都被分到 C1-C4 之一, 无 NULL。
- **AC2: 优先级正确**
  - **Given** 某股同时满足连板和重大事件
  - **Then** 应判定为 C4。
