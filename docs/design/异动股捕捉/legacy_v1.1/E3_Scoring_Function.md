## E3 · 综合评分与可解释性

### E3-S1 综合评分公式 (v1.1 增强版)

v1.1 引入了共振加成与反向抑制机制,公式如下:

```text
composite_score = (
    α · raw_score
    + β · score_l3_capital
    + γ · score_l4_emotion
    + ε · score_user_pref
    - ζ · score_dedup_penalty
    + μ · (resonance_level - 1) · 15        -- 共振加成
    - ν · (counter_signal_score / 100) · 20 -- 反向抑制
) * ads_market_state_daily.signal_reliability -- 市场状态可信度
```

#### 权重参数 (初始配置)
| 参数 | 对应项 | 初始权重 |
|---|---|---|
| α (Alpha) | 原始信号强度 | 0.40 |
| β (Beta) | L3 资金评分 | 0.25 |
| γ (Gamma) | L4 情绪评分 | 0.15 |
| ε (Epsilon) | 个人板块偏好 | 0.10 |
| ζ (Zeta) | 重复跟踪压制 | 0.10 |
| μ (Mu) | 共振等级加成 | 0.15 |
| ν (Nu) | 反向信号抑制 | 0.20 |

### E3-S2 评分增强逻辑

#### E3-S2-T1 共振加成
- 每个共振等级 (从 L2 开始) 会获得 15 分的原始增量,再乘以后置权重 μ。
- **目的**: 让多维印证的信号能够显著脱颖而出。

#### E3-S2-T2 反向抑制
- 根据 `counter_signal_score` (0-100) 按比例扣除最多 20 分的原始评分,再乘以后置权重 ν。
- **目的**: 自动降低有潜在风险 (如临近解禁、大股东减持) 信号的推送顺位。

#### E3-S2-T3 市场状态调节
- 当市场出现极端异常 (如系统性暴跌) 时,`signal_reliability` 会降至 0.6-0.8 之间。
- **目的**: 避免在系统性风险日产生误导性的推送。

### E3-S3 中文可解释性生成

系统根据信号的标签、共振维度及反向信号,自动生成一段自然语言描述,提升信号的可理解性。

#### 生成示例
> **寒武纪** 今日**首板涨停**。是**计算机**板块当日唯一首板。**主力资金净流入 1.20 亿**。涨停前**缩量蓄势**。隔夜**费城半导体指数上涨**。共振等级 **L4**(self、capital、sector、external 多维印证)。⚠️ 有基本面利空。
