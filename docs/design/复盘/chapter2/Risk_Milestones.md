# 第 2 章 · 风险与里程碑

## 风险与避坑指南

### 1. 数据质量类
- **申万行业 2021 改版**: 2021-12-13 之前的历史回补需注意 28 与 31 行业的差异。
- **概念板块“未来函数”**: 同花顺动态调整概念成分。做历史回溯时, 2024 年的数据用的是 2026 年的成分快照, 存在无法完全消除的偏差。
- **行业领涨股失真**: 小众行业(只有 2-3 只股)的领涨股参考价值低。

### 2. 技术性能类
- **MySQL 5.7 窗口函数**: 严禁直接使用 `ROW_NUMBER()`, 必须使用变量模拟(参考 E2)。
- **相关子查询性能**: 5 年历史回补计算时, 估值分位数 SQL 可能运行极慢, 建议按月/年分段执行。
- **JSON 字段限制**: `top_stocks` 使用 JSON 存储, 方便前端展示但不利于 SQL 复杂分析, 核心逻辑应基于 ODS 原始表。
