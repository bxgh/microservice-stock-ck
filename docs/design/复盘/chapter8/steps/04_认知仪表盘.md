# 第 8 章 · 认知训练系统 (第四步：认知仪表盘)

## E5 · 认知画像仪表盘 (Cognition Dashboard)

> 作为投资者,我希望看到自己 30/90/180 天的认知统计,以便了解擅长 / 盲区。

### E5-S1 关键指标 (后端聚合)

#### 准确率类
- **方向预判准确率**: `SUM(direction_hit) / COUNT(*)`, 基准 50%
- **量能预判准确率**
- **主线命中率**
- **高确信 vs 低确信的准确率分裂**: 理论上应该高确信更准

#### 决策质量类
- **4 象限分布饼图** (应得 / 蒙对 / 运气不佳 / 应输)
- **lucky_win 占比**: >30% 报警(运气成分过大)
- **平均判断依据数**: 应该 ≥ 3

#### 进步类
- **准确率趋势线**: 30/60/90 日趋势
- **避坑清单拦截率**: 规避错误的次数

### E5-S2 校准曲线 (Calibration Plot)

这是**最有价值的图**之一。横轴是用户的置信度(1-5),纵轴是实际命中率。

- **完美校准**: 45° 对角线
- **过度自信**: 曲线在对角线下方(典型问题)
- **过度保守**: 曲线在对角线上方

### E5-S3 月度认知报告 (自动生成)

每月 1 号自动生成, 结构如下：
- **整体准确率**: XX%
- **最大优势/盲区**: 识别量能/主线判断能力
- **风险提示**: 蒙对次数、大错次数
- **进步信号**: 证据数增加、情绪标签减少
- **建议聚焦**: 具体的行动改进建议

### E5-S4 验收标准

- **Given** 用户有 30 条以上预判,**When** 查看仪表盘,**Then** 展示校准曲线
- **Given** 每月 1 号,**When** 系统触发,**Then** 生成上月月度报告并 push
- **Given** 仪表盘加载,**When** 计算指标,**Then** 响应时间 < 800ms
