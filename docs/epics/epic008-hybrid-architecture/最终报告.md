## 1. 核心修复与功能增强说明

### 1.1 AkShare 接口 (8003) - [P0 修复]
- **龙虎榜修复**: 解决了旧版本返回 500 错误及数据过期的 Bug。现在支持自动匹配最新交易日数据。
- **数据清洗**: 引入了 `_clean_value` 装饰逻辑，彻底排除了 JSON 序列化时常见的 `NaN` 和 `Inf` 崩溃隐患。
- **财务摘要 (EPIC-002)**: 针对 AkShare 数据格式变动完成了适配。

### 1.2 BaoStock 接口 (8001) - [原生功能上线]
- **历史估值**: 正式上线原生历史 PE/PB/PS 接口，数据直接溯源至 BaoStock 数据库，不再依赖外部爬虫。
- **动态财务**: 盈利能力接口新增 `year` 和 `quarter` 动态参数。

### 1.3 PyWencai 接口 (8002) - [环境补全]
- **运行时支持**: Docker 镜像已集成 Node.js 运行时，彻底解决了内部库调用 JS 引擎失效导致的 `NoneType` 报错。

### 1.4 Stock Codes 接口 (8000) - [架构治理]
- **职责精简**: 物理移除了重复的 AkShare 路由与代理代码，大幅精简了镜像体积，确保该服务专注于“股票词典”职责。
