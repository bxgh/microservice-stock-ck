## 🔍 网络诊断记录

### 防火墙行为特征
1. **TCP 伪连通**：对 140+ 个随机 TDX IP 的 TCP 扫描均显示“成功”，但延迟极低（<1ms），推测是本地网关的应答，并非真实握手。
2. **DPI 深度过滤**：网关会解析 TDX 的私有握手协议。对于不在白名单内的 IP，在发送协议包时会立即触发 `Connection Reset (104)`。
3. **白名单策略**：目前仅放行了华为云公网段及个别大型券商机房（如海通）的 7709 协议流量。
