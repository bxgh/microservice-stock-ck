# Microservice Stock Web

> 微服务股票系统 - 通用Web前端应用

## 🚀 项目介绍

这是微服务股票系统的通用Web前端，提供跨服务的统一管理界面。

**主要功能:**
- 📊 系统概览和监控
- 🔗 微服务状态监控
- 📈 数据可视化展示
- 👥 用户管理和权限控制

## 🛠️ 技术栈

- **框架**: Vue 3.4+ with Composition API
- **语言**: TypeScript 5.0+
- **构建工具**: Vite 5.0+
- **UI库**: Element Plus
- **状态管理**: Pinia 2.x
- **路由**: Vue Router 4

## 📦 快速开始

\`\`\`bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
\`\`\`
