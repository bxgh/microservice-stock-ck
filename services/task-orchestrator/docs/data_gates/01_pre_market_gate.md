# 数据门禁 (Gate-1): 盘前数据就绪校验

## 1. 目标
在每个交易日开盘前 (08:00-09:20)，自动完成：
1. **昨日数据完整性校验** - 核心资产 K线、分笔覆盖率检查
2. **自动补采触发** - 数据不足时调度补采任务，确保开盘前数据基底完备
3. **状态准入** - 决定当日策略扫描是否具备高质量数据前提

## 2. 执行逻辑
- **调度时间**: 每个交易日 `08:00`
- **校验对象**: T-1 (上一个交易日) 数据
- **核心指标**:
    - K线覆盖率 > 98%
    - 分笔覆盖率 > 95%

## 3. 补采机制
- 若指标不达标，自动触发 `daily_kline_sync` (全量) 或 `repair_tick` (差异补采)。
- 补采完成后，Gate-1 会进行二次校验并更新报告。

## 4. 告警通知
- 发送“盘前就绪报告”至企业微信。
- 只有 Gate-1 通过，当日常驻采集服务方可视为处于“安全环境”。
