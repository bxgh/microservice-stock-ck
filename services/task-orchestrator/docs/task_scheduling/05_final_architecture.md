## 服务清单

| 服务 | 来源 | 运行模式 | 目录 | 职责 |
|:-----|:-----|:---------|:-----|:-----|
| **gsd-api** | get-stockdata 拆出 | 长驻 | services/gsd-api/ | 数据查询 |
| **gsd-worker** | get-stockdata 拆出 | 临时 | services/gsd-worker/ | 同步+质量+修复 |
| **gsd-shared** | 新建 | 包 | libs/gsd-shared/ | 共享模型 |
| **task-orchestrator** | 升级 task-scheduler | 长驻 | services/task-orchestrator/ | 任务编排 |
| **get-stockdata** | 保留 | 长驻 | services/get-stockdata/ | 兼容层 |

## 每日任务流程

```
18:00  task-orchestrator 检测: 今天是交易日？
       │
       ├─ 是 ──▶ 启动 gsd-worker 容器 (并行分片)
       │              │
       │              ├── shard-0: 股票 000001-001500
       │              ├── shard-1: 股票 001501-003000
       │              ├── shard-2: 股票 003001-004500
       │              └── shard-3: 股票 004501-XXXXXX
       │              │
       │              └── 同步完成 (~2.5分钟)
       │
       ├──▶ 启动 gsd-worker --job=quality-check
       │         │
       │         ├── 完整性检测
       │         ├── 缺失 < 50 只？自动修复
       │         └── 完成
       │
       ├──▶ 销毁所有 worker 容器
       │
       └──▶ 发送通知: 今日数据同步完成
```

## 技术决策 (最终)

| 决策点 | 选择 |
|:-------|:-----|
| 服务数量 | 2+1 (api + worker + orchestrator) |
| sync/quality | 合并为 gsd-worker |
| 原 get-stockdata | 保留作为兼容层 |
| worker 运行模式 | 临时任务 (用完销毁) |
| 并行策略 | 分片 (4 容器) |
