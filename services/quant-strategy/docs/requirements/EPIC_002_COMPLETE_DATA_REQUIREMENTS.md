## 九、数据质量要求

### 9.1 准确性
- 财务数据必须与公司官方报告一致
- 市场数据需与交易所数据一致
- 对可疑数据进行标记 (如突然10倍变化)

### 9.2 完整性
- 所有必填字段不能为空
- 缺失数据明确标记为 `null`
- 历史数据至少覆盖2年 (8个季度)

### 9.3 一致性
- 统一使用亿元作为金额单位
- 统一使用 YYYY-MM-DD 日期格式
- 统一使用申万或证监会行业分类

## 附录: 计算公式参考

### ROE (净资产收益率)
```
ROE = 净利润 / 净资产 × 100%
```

### ROIC (投入资本回报率)
```
ROIC = NOPAT / 投入资本 × 100%
其中: 投入资本 = 净资产 + 有息负债
```

### 毛利率
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```

### 营收增速 (YoY)
```
营收增速 = (本期营收 - 去年同期营收) / 去年同期营收 × 100%
```

### 营收 CAGR (3年)
```
CAGR = (期末值 / 期初值)^(1/年数) - 1
```

### PE Band 估值
```
当前 PE 分位数 = (当前 PE - 历史最小 PE) / (历史最大 PE - 历史最小 PE)
```

### DCF 估值 (简化)
```
企业价值 = Σ(未来现金流 / (1 + 折现率)^年数)
```
