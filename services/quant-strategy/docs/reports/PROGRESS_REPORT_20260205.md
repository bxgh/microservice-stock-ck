## 🏗️ 核心产出

### 1. 数据清洗 (DataCleaner)
- **对应 Story**: 002.01
- **功能**:
    - 将 3秒 Level-1 快照清洗为 1分钟标准 OHLCV 序列 (240个时间点)。
    - 处理了涨跌停、集合竞价、停牌等边界情况。
    - 实现了前向填充与异常值过滤。
- **验证**: 2026-02-03 全天成交量与 Snapshot 总量误差 < 0.1%。

### 2. 基础特征 (BasicFeatureEngine)
- **对应 Story**: 002.02
- **功能**:
    - **Vector A**: 基于 Lee-Ready 算法的主动买卖强度。
    - **Vector B**: 5档加权 OBI (Order Book Imbalance)。
    - **Vector C**: 分钟级对数收益率。
- **技术点**: 解决了 Decimal 类型计算兼容性问题。

### 3. 交易规模 (TradeSizeEngine) [P0]
- **对应 Story**: 002.03
- **功能**:
    - 基于金额的四档分类 (散户/中单/大单/超大单)。
    - **LOR**: 大单占比，有效识别主力资金。
    - **NLB**: 机构净买入金额。
    - **RID**: 机构/散户背离度 (量化指标)。
- **效果**: 成功捕捉到尾盘集合竞价的大单特征 (LOR=1.0)。

### 4. 流动性风控 (LiquidityGatekeeper)
- **对应 Story**: 002.04
- **功能**:
    - **VPIN**: 动态体积桶 + BVC 估算，精准衡量毒性流 (验证值 ~0.6)。
    - **Kyle's Lambda**: 分钟级价格冲击系数 ($10^{-6}$ 量级)，准确量化市场深度。
