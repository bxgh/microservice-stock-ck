# EPIC-002: 长线资产配置系统 (70% 仓位)

**版本**: v1.1
**状态**: 📅 规划中
**优先级**: P1
**依赖**: EPIC-001

## 📋 概述
实现文档中定义的“核心长线仓”逻辑，专注于高确定性、低估值成长股及高分红红利资产的筛选与配置建议。目标年化收益 15-25%。
引入 **Alpha 4D-Scoring (四维评分模型)**，结合“机构审美”与“实战避雷”逻辑，构建严谨的长线股票池。

## 🎯 目标
- 自动化筛选符合行业配置建议的标的。
- 实施 **Alpha 4D-Scoring** 评分体系（Valuation + Quality + Growth + Holding）。
- 执行严格的 **负面清单剔除 (Veto List)** 机制，排除财务雷区。
- 生成季度调仓建议和分批建仓计划。

## 📚 User Stories

### Story 2.1: 负面清单排雷系统 (Risk Veto)
> *“先生存，后发展。长线投资最大的风险是本金永久损失。”*
- **目标**: 在打分前先行剔除存在致命风险的标的。
- **任务**:
  - 实现一票否决逻辑，命中即剔除：
    - **ST/退市风险**: 标记为 ST, *ST 或处于退市整理期。
    - **商誉暴雷**: `商誉 / 净资产 > 30%`。
    - **质押危机**: `大股东质押率 > 50%`。
    - **监管黑名单**: 近 12 个月有立案调查或重大违规记录。
    - **财务异常**: `经营性现金流 / 净利润 < 0.5` (且非银行股)。

### Story 2.2: 行业估值与安全边际 (Valuation Safety)
- **目标**: 评估“价格”是否合理，规避价值陷阱和拥挤赛道。
- **任务**:
  - **历史分位**: 计算 PE-TTM / PB 在过去 5/10 年的分位数（<20% 低估，>80% 高估）。
  - **PEG 模型**: 针对成长股（AI/医药），计算 `PE / 净利润增速` (PEG < 1 为优)。
  - **股息率差**: 计算 `股息率 - 十年期国债收益率`， >3% 视为具备极高配置价值（险资逻辑）。

### Story 2.3: 质量与护城河评分 (Quality Moat)
- **目标**: 识别“真核心资产”，剔除伪成长。
- **任务**:
  - **ROE 稳定性**: 过去 3 年平均 ROE > 15%，且标准差 < 5%（通过杜邦分析剔除高杠杆驱动）。
  - **自由现金流**: 关注 `FCF` 为正的企业。
  - **研发投入**: 针对科技股，考察 `研发支出 / 营收` 占比及趋势。

### Story 2.4: 景气度与机构动向 (Momentum & Holding)
- **目标**: 捕捉“未来”增长和“聪明钱”共识。
- **任务**:
  - **业绩加速**: 季度营收/利润增速环比提升。
  - **分析师预期**: 监控近期分析师上调盈利预测的次数（pywencai 语义分析）。
  - **北向资金**: 连续 N 日净流入或持仓占比创历史新高。
  - **筹码集中**: 股东户数连续减少。

### Story 2.5: 核心赛道选股策略 (Sector Specific)
- **目标**: 针对不同赛道应用差异化打分权重。
- **任务**:
  - **策略 A (红利低波)**: 权重倾斜于 `股息率` (40%) + `PB` (30%) + `现金流` (30%)。
    - 标的：中国神华、银行、高速公路。
  - **策略 B (核心成长)**: 权重倾斜于 `PEG` (30%) + `增速` (30%) + `北向资金` (20%) + `ROE` (20%)。
    - 标的：AI算力、创新药、新能源龙头。

### Story 2.6: 组合构建与动态调整
- **目标**: 构建 8-10 只股票的中线组合，控制单行业暴露。
- **任务**:
  - **组合优化**: 限制单一行业权重 < 30%，确保赛道分散。
  - **建仓计划**: 生成 `30%底仓 + 30%加仓 + 40%决胜仓` 的分批买入信号（结合回撤幅度触发）。
  - **季度再平衡**: 每季度末重新运行评分，对排名下降的标的生成“调出”建议。
