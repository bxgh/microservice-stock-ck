# 日内动量与隔夜跳空策略

> **优先级**: P1 - 高优先
> **理论来源**: Reading University, Alpha Architect, 约翰内斯堡交易所实证研究

## 1. 核心假设

### 1.1 隔夜动量假设

**假设**: 动量收益的绝大部分来自隔夜，而非日内。

**实证支持**:
- Alpha Architect研究：动量策略95%的超额收益发生在隔夜
- 隔夜Alpha 0.95% vs 日内Alpha 0.11%（统计不显著）

**原因解释**:
- 机构投资者倾向于盘后/盘前交易
- 重大信息多在非交易时段发布
- 日内交易噪音大，信息效率低

### 1.2 跳空均值回归假设

**假设**: 大幅跳空后，日内存在均值回归。

**实证支持**:
- 约翰内斯堡交易所研究：高开后日内回落显著盈利
- 逻辑：过度反应 → 日内修正

---

## 2. 关键概念

### 2.1 隔夜收益 (Overnight Return)

$$R_{overnight} = \frac{Open_{today} - Close_{yesterday}}{Close_{yesterday}}$$

### 2.2 日内收益 (Intraday Return)

$$R_{intraday} = \frac{Close_{today} - Open_{today}}{Open_{today}}$$

### 2.3 跳空幅度 (Gap)

$$Gap = R_{overnight}$$

- Gap > 0：高开
- Gap < 0：低开

---

## 3. 策略逻辑

### 3.1 隔夜动量策略

**逻辑**: 捕捉隔夜信息优势。

| 条件 | 操作 |
|------|------|
| 前日收盘买入 | 持仓过夜 |
| 当日开盘卖出 | 获取隔夜收益 |

**适用场景**:
- 信息驱动型股票
- 机构重仓股

### 3.2 跳空回归策略 (Fade the Gap)

**逻辑**: 大幅跳空通常过度反应，日内回归。

| 条件 | 操作 |
|------|------|
| Gap > +2% | 开盘做空（等待回落） |
| Gap < -2% | 开盘做多（等待反弹） |
| 收盘平仓 | 获取日内回归收益 |

**风险提示**:
- 突破型跳空（Breakaway Gap）可能延续而非回归
- 需结合成交量判定跳空类型

### 3.3 日内动量传导

**逻辑**: 开盘30分钟收益可预测收盘30分钟收益。

**实证公式**:
$$R_{last\_30min} = \alpha + \beta \times R_{first\_30min} + \epsilon$$

研究表明β显著为正，可用于日内趋势判断。

## 4. 跳空类型识别

### 4.1 分类标准

| 跳空类型 | 成交量特征 | 后续走势 | 策略 |
|----------|------------|----------|------|
| 普通跳空 | 量平 | 回归 | Fade |
| 突破跳空 | 量增 | 延续 | Follow |
| 竭尽跳空 | 巨量 | 反转 | Fade |
| 缺口回补 | 量缩 | 震荡 | 观望 |

### 4.2 成交量确认

$$VolumeRatio = \frac{开盘30分钟成交量}{过去20日同时段均量}$$

- VolumeRatio > 2.0 → 突破型跳空可能性高
- VolumeRatio < 0.8 → 普通跳空，回归概率高

## 5. 与原框架集成

### 5.1 增强集合竞价分析

原方案仅提取集合竞价特征作为补充标签，可增强为独立信号源：

| 特征 | 信号 |
|------|------|
| Gap > 2% 且 VolumeRatio < 1.5 | 回调买入机会 |
| Gap > 2% 且 VolumeRatio > 2.0 | 追涨（趋势延续） |
| Gap < -2% 且无明确利空 | 超卖反弹机会 |

### 5.2 补充盘后分析的时效性

原方案为盘后分析，信号次日才能使用。日内动量策略可提供**日内实时信号**：

- 观察开盘30分钟走势
- 预测收盘30分钟方向
- 日内完成交易闭环

## 6. 信号生成

### 6.1 隔夜持仓信号

**买入条件**（尾盘买入，次日开盘卖出）:
- 股票处于Cluster龙头位置
- 前一日大单净买入为正
- 当日无重大利空公告

### 6.2 跳空回归信号

**做多条件**:
- Gap < -2%
- VolumeRatio < 1.5
- 无明确利空（需NLP或人工判定）

**做空条件**（A股无法做空，可减仓）:
- Gap > +3%
- VolumeRatio < 1.5
- VPIN > 0.4（流动性趋紧）

### 6.3 日内动量信号

**买入条件**:
- 开盘30分钟收益 > +1%
- 成交量放大
- 盘口买压 > 卖压

**预期**:
- 收盘30分钟大概率延续上涨

## 8. 回测验证要点

### 8.1 验证假设

1. 隔夜收益是否占总收益的主要部分？
2. 大幅跳空后日内回归的胜率？
3. 开盘30分钟与收盘30分钟的相关性？

### 8.2 分层回测

| 分层条件 | 观察指标 |
|----------|----------|
| Gap > 2% | 日内回落幅度 |
| Gap < -2% | 日内反弹幅度 |
| VolumeRatio > 2 | 趋势延续概率 |
| VolumeRatio < 1 | 均值回归概率 |

## 9. 参数配置

```yaml
intraday_momentum:
  # 跳空阈值
  gap:
    large_up: 0.02    # 2%
    large_down: -0.02
    extreme_up: 0.03  # 3%
    extreme_down: -0.03
    
  # 成交量确认
  volume:
    ratio_threshold: 1.5
    lookback_days: 20
    
  # 日内时段
  periods:
    first_30min: [0, 29]    # 分钟索引
    last_30min: [210, 239]
```

## 10. 风险提示

### 10.1 策略风险

| 风险类型 | 描述 | 缓解措施 |
|----------|------|----------|
| 趋势延续 | 跳空突破不回归 | 成交量确认 |
| 黑天鹅 | 重大利空导致持续下跌 | 止损设置 |
| 流动性 | 开盘流动性差，滑点大 | 避开小盘股 |

### 10.2 A股特殊性

- T+1制度：无法当日做空
- 涨跌停制度：极端跳空受限
- 集合竞价规则：开盘价形成机制不同
